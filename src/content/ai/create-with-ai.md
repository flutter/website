This guide covers how to integrate AI into your development process to write
code faster, and how to build generative AI features directly into your
Flutter apps.

## Develop with AI

AI tools can act as a powerful assistant in your daily development workflow, helping you write code faster, analyze complex errors, and reduce boilerplate. 

The tools listed below can be integrated into your IDE or terminal:

*   **[Antigravity](https://antigravity.google/)**: An in-IDE AI agent that can plan tasks, read and write code, run terminal commands, and verify its own changes for you.
*   **[Gemini Code Assist](https://codeassist.google/)**: An AI-powered collaborator for IDEs like VS Code and Android Studio that provides real-time code completion, in-editor chat, and error explanation.
*   **[Gemini CLI](https://geminicli.com/)**: A command-line AI tool to quickly scaffold new Flutter widgets, use automated rules, and interact with MCP servers.
*   **[Dart and Flutter MCP server]({{site.dart-site}}/tools/mcp-server)**: A Model Context Protocol (MCP) implementation that connects AI tools directly to the Flutter widget tree, pub.dev, and your runtime environment.
*   **[Rules for Flutter and Dart](/ai/ai-rules)**: Configuration files used to provide project-specific context and enforce best practices for your underlying LLMs.

## Build AI-powered apps

Integrating generative AI into your Flutter application unlocks new user experiences, allowing your app to support natural language processing, intelligent orchestration, and dynamic content generation.

Use the following software development kits (SDKs) and toolkits to build AI features:

*   **[Firebase AI Logic client SDKs]({{site.firebase}}/docs/ai-logic)**: The official Firebase SDK to securely access Gemini and Imagen models directly from your Flutter client.
*   **[GenUI SDK for Flutter](/ai/genui)**: An experimental orchestration layer that coordinates communication between users, AI agents, and custom interactive Flutter widgets.
*   **[Flutter AI Toolkit](/ai/ai-toolkit)**: A collection of pre-built UI widgets (like chat interfaces) designed specifically to help you quickly build AI-powered features in Flutter.
