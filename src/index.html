빠른 개발

상태를 저장하는 Hot Reload를 이용하여 앱을 생생하게 재현해보세요. 커스터마이징을 완전히 지원하는 다양한 위젯을 이용하여 몇 분만에 네이티브 인터페이스를 구축해보세요.

{% asset homepage/icon-ui.svg alt='Expressive and flexible IU icon' %}

표현력 있고 유연한 UI

최종 사용자의 경험에 초점을 맞춰 신속하게 기능을 제공하세요. 계층형 아키텍처를 통해 완벽한 커스터마이징을 할 수 있기 때문에, 매우 빠른 렌더링과 표현력 있고 유연한 디자인을 구현이 가능합니다.

{% asset homepage/icon-performance.svg alt='Native performance icon' %}

네이티브 수준의 성능

Flutter 위젯은 플랫폼별 차이를 통합하여 스크롤, 내비게이션, 아이콘, 글꼴 등을 iOS와 안드로이드 모두에서 네이티브 수준의 성능으로 제공합니다.

{% asset homepage/icon-development.svg alt='Fast development icon' %}

빠른 개발

Flutter의 hot reload는 쉽고 빠른 실험, UI 빌드, 기능 추가, 빠른 버그 수정을 도와줍니다. 에뮬레이터, 시뮬레이터, iOS 기기, Android 기기에서 상태를 잃지 않는 1초 내 리로드를 경험해보세요.
